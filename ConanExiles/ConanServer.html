INI Settings Legend

How To Read

Headers:

  • * = Recommended Settings
  • 1 = Civilized Preset Setting
  • 2 = Decadent Preset Setting
  • 3 = Barbaric Preset Settings
    •

    Colors:

  • White = INI Changeable Only
  • Gray = In-Game Changeable
  • Purple = Unknown Settings
  • Red = Broken Settings

    "Dedicated Server Launcher"
    All settings need prefix: "DedicatedServerLauncher" in place of "..."
    ...AutoRestartEnabled=-FalseAutomatically restart the Server
    ...AutoRestartPerDay=-1Amount of times the Server should restart per day
    ...AutoRestartMinimumUpTime=-352800A 24 hours time selection control used to define the minimum uptime of the server before a restart (Default is 2 hours)
    ...AutoRestartTime=-367200A 24 hours time selection control used to define the time of the day when the server will restart (Default is 6 AM)
    ...AutoRestartWarningMessage1="Server shutting down in 10 Minutes."""First Server restart Warning Message
    ...AutoRestartWarningMessage2="Server shutting down in 5 Minutes."""Second Server restart Warning Message
    ...AutoRestartWarningMessage3="Server shutting down in 2 Minutes."""Last Server restart Warning Message

    ...AutoRestartWarningDelay2=-345900Display Second Warning X amount of Minutes before restart. (Default is 5 Minutes)
    ...AutoRestartWarningDelay3=-345720Display Last Warning X amount of Minutes before restart. (Default is 2 Minutes)
    ...AutoUpdateStrategy=-1Automatic Update method of Server.
    0: Don't Auto-Update on Restart
    1: Auto-Update on Restart
    2: Check for Updates and Restart
    ...BackgroundCheckDelay=-345900Delay before the Auto-Update Check. (Default is 5 Minutes)
    ...BackupEnabled=TrueFalseAutomatically make a Back-Up of the Server on Server Shutdown

    ...ExtraCommandLineParameters=-""Extra Server Commandlines on Server Start

    ...ServerRestartedMessage=-""Discord Message to send, when the Server is Restarting

    ClientCatchUpTime=2345In order to create a smooth starting experience for new players, Conan Exiles has the ability to force a client PC to start at a certain time of day, regardless of the server time. If this setting is applied, new characters will start at the specified time, and then play at that time of day until the server “catches up” to them.
    We recommend not setting this to the darkest hours of the night, unless you are a sadistic person
