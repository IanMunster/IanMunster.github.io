    • Admin Tools

    Action Controls Description
    Open Admin Panel [CTRL]+[SHIFT]+[C]
    -or-
    [SHIFT]+[INSERT]    		 Opens the Admin Panel GUI
    Open Console Commands Interface [INSERT]
    -or-
    [~]    		 Opens the Console Commands Interface, where you can type Console Commands
    Open Full Console Commands Interface Double press [INSERT]
    -or-
    Double press [~]    		 Open the Full Console Commands Interface, show history of parsed commands
    Get Precise Coördinates [R-CTRL]+[R-SHIFT]+[R-ALT]+[L] Opens GUI that readout your Coördinates in the Gameworld
    Kill / Destroy Target [SHIFT]+[DELETE] Kills or Destroys a target under your Cursor:
    • Removes a single Building Part
    • Kills a single NPC
    • Does NOT delete World-Chunks, nor Campfires, Forges etc.
    Kill / Destroy Multiple Targets [CTRL]+[SHIFT]+[DELETE] Kills or Destrots Multiple targets under your Cursor, with a larger radius:
    • Removes multiple Building Part
    • Kills multiple NPC, also Pets
    • Does delete World-Chunks
    Infinite Building Item Stacks [CTRL]+[ALT]+[SHIFT]+[F10] Infinite Building Item Stacks in your Inventory, you must have at least one quantity per related item in your inventory
    Start Sandstorm [CTRL]+[ALT]+[SHIFT]+[X] Starts a Sandstorm, it will take a few minutes to reach your position
    Command Description
    "Admin Manipulation"
    MakeMeAdmin Enters Admin mode. Triggers Admin Rights Granted
    MakeMeNormal Exits Admin mode
    God Toggle God mode. Invulnerable to all damage, knockbacks, status effects etc. Only exception is the Cursewall
    Cloak Toggle Cloak Mode. Invisible to NPCs, even when attacking them
    Invisibility Toggle Invisibility Mode. Makes your character invisible
    NoSprintCost Toggle No Sprinting Stamina Cost. No stamina drain while sprinting. Not unlimited stamina
    Fly Grants Fly mode. Fly mode has two different poses, "Fly" and "FlyB", which are occasionally switched
    Ghost Grants Ghost mode.
    • Grants Fly mode (See Fly)
    • Grants Noclip, you can pass through objects
    • Building is possible, but building pieces will not snap to other building pieces
    Walk Exits the Fly or Ghost mode
    "Server Manipulation"
    ToggleDebugHUD Admin mode not required. Open Player Data GUI.
    • Displays current location, in the lower left of the screen
    • Displays other data, including ping, in the upper right of the screen
    GetServerHealth Toggle the Full Console Commands Interface to see the Output:
    Lists server statistics in the console
    • FPS; Frames Per Second of the Server
    • Number of Players Connected to the Server
    • Number of Spawned Actors in the Server
    • Number of Player Bases, including detached foundations
    • Number of Building-Parts in the Server. Parts include: Foundations, Walls, Roofs. Possibly also includes: Chests, CraftingStations etc.
    • LOD Count; Level of Detail
    GetAllServerSettings Toggle the Full Console Commands Interface to see the Output:
    Displays a list, of all available server settings you can modify, in the console.
    SetServerSetting "[ServerSettingName]" [Value] Quotations are required.

    Allows you to modify the Server Settings inside ServerSettings.ini while in-game. Includes server settings not changeable in Server Settings tab
    (*) For admins using Rcon, Server Settings can be set remotely via this command as well
    FlushLog (!) As of May, 2020, this command does not work) Clears the Server Event log. The log can be viewed by pressing ESC then selecting Event Log
    "Player Manipulation"
    ShowPlayers Lists connected players, Steam names and IDs
    PrintPlayerInfo [(*)PlayerName/SteamID] Toggle the Full Console Commands Interface to see the Output:
    Gives you the specific players Steam name and in-game character name
    (*): You can enter a players character name to get their Steam name -or- you can enter the Steam name to get their character)
    ViewPlayer [PlayerName] Moves the camera to the Player character
    ViewSelf Moves the camera back to your own character
    Teleport Teleports yourself
    • to a surface under the cursor
    • when targeting the air;
      • Vertically: Higher up in the sky (Currently, targeting the sky teleports you to 0,0,0.)
      • Horizontally: within ~1 Map chunk from the Cursewall.
    • to a targeted player: ViewPlayer + Teleport + ViewSelf
    TeleportPlayer [(*)PlayerName] [X] [Y] [Z] Teleports yourself to the specified coordinates X[float], Y[float] and Z[float] are the X, Y and Z coordinates on the map
    (*): Teleport another player)
    TeleportToPlayer [PlayerName] Teleports yourself to the specified player
    SummonPlayer [PlayerName] Teleports a player to your current position. May be used to fix stuck/glitched players
    BroadcastMessage [Message] Sends a message to all online players via a pop-up box which needs to be clicked on to close (can annoy players as it interrupts gameplay, especially when in combat)
    SpawnItem [ItemID] [Value] Spawns the specified item in the specified quantity
    LearnEmote [Emote] Learn an Emote. Type /emote [emote] in chat or hold R to bring up the Emote menu
    "World Manipulation"
    PlayerCanBuildEverywhere [PlayerName] Allows building where building is disabled for the specified player
    BuildingIndentify Toggle the Full Console Commands Interface to see the Output:
    Shows a building information under the cursor:
    • Displays Buildable ID
    • Displays Owner ID
    • Displays Owner Name
    • Displays Position X, Y, Z
    • Displays Actor (Developemnt-Kit path to Buidable)
    Buildable ID: Can be used to determine if structures are connected for decay purposes If the player is disconnected, the Buildable ID will change.
    BuildingDestroy [BuildingID] Destroys entire building with the BuildingID as determined above
    DamageTarget [Value] Deals damage to the object or NPC in under your cursor.
    Will not hurt creatures. They are pulled towards you instead. If the [Value] is eg. 9999 it will instantly destroy a single targeted building piece. It is recommended to use Destroy/Kill (Controls section) instead
    DestroyTarget (!) Use with caution Removes (almost) entire player built structures. Temporarily destroys all objects of the same kind, eg. trees. Includes targeted parts of the map. Players will fall into and die in the void here. Singleplayer requires a relog and servers require a restart to undo this. It is recommended to use Destroy/Kill (Controls section) instead
    "Follower Manipulation"
    LevelFollowerUpTo [Value] Levels follower to level [Value] (20 max)
    GetFollowerStat [Value] Get the Current starts of the follower. Requires a parameter at [Value], such as "DamageModifierMelee" or "DamageModifierRanged"
    "Purge Manipulation"
    StartPlayerPurge Forces the Start of a Purge, when the Purge meter is Full
    StartNextWave Starts the next wave of the currently active Purge
    StartNextPurgePhase Forces a new Purge to start
    EndPurge Ends the currently active Purge
    FillPlayerPurgeMeter (?)Fills the Purge Meter of a Player(?)
    FillPlayerClanPurgeMeter (?)Fills the Purge Meter of a Clan(?)
    FillClanPergeMeter (!)The command is currently mispelled. Fills the Purge Meter of a Clan in Singleplayer only
    FillAllClanPurgeMeter (?)Fills the Purge Meter of all Connected Clans(?)
    EmptyPlayerClanPurgeMeter (?)Sets the Players Purge-Meter to 0(?)
    EmptyAllClanPurgeMeters (?)Sets all of the Connected Clans Purge-Meter to 0(?)
    "Game Quality Manipulation"
    r.ShadowQuality [Value] Toggles shadow transparency: 0 = Disables shadows 1 = Default
    p.clothphysics [Value] Toggles dynamic cloth movement speed.: 0 = Disables dynamic cloth movement 1 = Default

    "Dedicated Server Launcher"
    All settings need prefix: "DedicatedServerLauncher" in place of "..."
    ...AutoRestartEnabled= - False Automatically restart the Server
    ...AutoRestartPerDay= - 1 Amount of times the Server should restart per day
    ...AutoRestartMinimumUpTime= - 352800 A 24 hours time selection control used to define the minimum uptime of the server before a restart (Default is 2 hours)
    ...AutoRestartTime= - 367200 A 24 hours time selection control used to define the time of the day when the server will restart (Default is 6 AM)
    ...AutoRestartWarningMessage1= "Server shutting down in 10 Minutes." "" First Server restart Warning Message
    ...AutoRestartWarningMessage2= "Server shutting down in 5 Minutes." "" Second Server restart Warning Message
    ...AutoRestartWarningMessage3= "Server shutting down in 2 Minutes." "" Last Server restart Warning Message
    ...AutoRestartWarningDelay1= - 346200 Display First Warning X amount of Minutes before restart. (Default is 10 Minutes)
    ...AutoRestartWarningDelay2= - 345900 Display Second Warning X amount of Minutes before restart. (Default is 5 Minutes)
    ...AutoRestartWarningDelay3= - 345720 Display Last Warning X amount of Minutes before restart. (Default is 2 Minutes)
    ...AutoUpdateStrategy= - 1 Automatic Update method of Server.
    0: Don't Auto-Update on Restart
    1: Auto-Update on Restart
    2: Check for Updates and Restart
    ...BackgroundCheckDelay= - 345900 Delay before the Auto-Update Check. (Default is 5 Minutes)
    ...BackupEnabled= True False Automatically make a Back-Up of the Server on Server Shutdown
    ...BackupPath= "%ServerRoot% /DedicatedServer_BackUps/" "" Location to save the automatic Back-Up
    ...BackupTrimmingEnabled= True False Enable the limiting option to only keep a certain number of days of Back-Ups
    ...BackupTrimmingLimit= - 7 X Number of days of Back-Ups to keep
    ...ModList= - "" The Mods the Server is Using, Either with a SteamWorkshop ID or a Local Pathname
    ...ProcessPriority= 2 0 Server Processor Priority.
    0 = Normal
    1 = Above Normal
    2 = High

    ...SteamCMDDisabled= - False Disables the Steam CMD, Function Unknown

    ...DiscordEnabled= True False Enable the Discord Automated Message Bot
    ...DiscordTimeStampsEnabled= True False Display a Timestamp of the Automated Message
    ...DiscordWebHooks= - "" The Discord Hook link to connect to the Discord Server
    ...DiscordUsername= - "" The Discord Automated Message Bot's Name
    ...ServerReadyMessage= - "" Discord Message to send, when the Server is Online

    ...ServerShutdownMessage= - "" Discord Message to send, when the Server is Shutting Down

    Setting * 1 2 3 Description

    Note, this multiplier is applied in addition to any settings changes below

    Server Messaging System

    You can use the three warming messages to automatically send a message to all the connected players:

      Rcon needs to be enabled (with a valid password, port, etc…)
      The time is the number of hours or minutes relative to the expected restart time

    Note that since the delay accepts up to a 24 hours range, you can creatively use the feature to inform your players of regular events, advertise things, etc… the only limit is your imagination and the amount of bugs you may find when playing with it.

    Messages can also contain “macros”, which are symbols that get expanded when displayed.
    As of now there are only two available macros, but they are practical to send useful information to the players:

      {servername} - expanded to what you have set up in the Server Name field
      {steamconnect} - expanded to a connection string players can click on to connect
      {steamconnect_nopassword} - same thing, but without the password being displayed
      {reason} - expanded to display the reason for the server restart (mod update, automated, etc…)
      {ip} - replaced by the external IP address of the server
      {game_port} - replaced by the value of the game port
      {steam_port} - replaced by the value of the steam query port
      {rcon_port} - replaced by the value of the rcon port
      {server_password} - replaced by the server password
      {admin_password} - replaced by the admin password (beware!)
      {rcon_password} - replaced by the rcon password (beware!)

    Example setting the “Ready” message to:

      The {servername}'s server is now open, please join on {steamconnect}

    will allow your Discord player to see the following clickable message:

      The Toolguy’s server is now open, please join on steam://connect/123.231.132.213:27015/pass

    Remember: The message sending feature requires Rcon and/or Discord WebHook to be enabled!
